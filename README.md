## рЯУМ Project Description

The Supply Chain Tracker is a decentralized application built using Soroban smart contracts on the Stellar network. It enables transparent and tamper-proof tracking of goods as they move through different stages of a supply chain.

This project helps businesses ensure authenticity, reduce fraud, and improve traceability by recording every step of a product's lifecycle on-chain.
